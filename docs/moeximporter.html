161        """Internal method for standartize https-queries to MOEX ISS.  
162

167

174        _pparams: dict, optional

176            parameter and value is its string value. These parameters are
177            used in the url string and should be replace with values.  
178            Example:  
179            in the url template

182            ticker.  {'__SECCODE___':'GAZP'} is passed to the method.
183        _params: dict, optional
184            Dictionary of pairs id:value where id is a string id of the
185            parameter and value is its value. The value type depends on the
186            request and automatically converted for the url in this method.  
187            Example:  
188            request  
189            https://iss.moex.com/iss/securities.json  
190            may use 2 parameters in some cases: start and is_trading  
191            https://iss.moex.com/iss/securities.json?start=0&is_trading=1  
192            the argument should be {'start': 0, 'is_trading': '1'}
193                
194        Returns
195        -------
196        array_like

198            This array should be parsed and converted for usage.

202        _values = self.base_values.copy()

206            _rpp = _mr['postfix_params']
207            if _pparams:
208                for _ppkey in _pparams:
209                    if _ppkey in _rpp:

211            _rp = _mr['params']
212            if _params:
213                for _pp in _params:
214                    if _pp in _rp:

216            _data = urllib.parse.urlencode(_values)
217            _url += f'?{_data:s}'
